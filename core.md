下面是为后续 AI 和团队参考的“预测市场进入中国的核心创意与方针”（简洁、可复用、面向决策者与开发者）。

核心一句话（电梯陈述）
- 我们是一个“零真钱、以声誉为回报”的模拟预测平台，用量化的预测与复盘把知识社区、行业专家与企业决策连接起来——通过可审计的模拟市场培养能力、展示专业并为企业提供可信的群体智力信号。

使命（Mission）
- 通过安全、透明且合规的模拟预测，让个人和组织提升判断力、量化知识与建立可迁移的专业声誉。

不变的五条基本原则（Non‑negotiables）
1. 绝对无真钱/无加密/无可兑换货币。任何积分均不可购买、不可提现、不可转让。  
2. 声誉优先：用户回报以“可公开的能力指标/徽章/证书/企业导出”形式体现，而非金钱。  
3. 合规与可审计：所有市场定义、交易与结算保留不可篡改日志，并备律师书面意见作为合规证据。  
4. 内容安全：禁止政治、司法、国家安全、重大公共卫生、金融衍生品、体育博彩式题材及侵犯个人隐私的市场。  
5. 公平与防作弊：实行实名/阶梯 KYC、速率与暴露上限、行为分析与人工复核相结合。

核心价值主张（核心用户为何来）
- 个人：练习概率思维、获得可展示的专业声誉、参与高质量讨论与复盘。  
- 社区/媒体：产生可交互且有数据支撑的知识内容（可用于内容变现与流量）。  
- 企业/机构：获得经过聚合与匿名化的群体预判信号，辅助决策与人才识别。

核心产品思路（保持抽象以适应玩法变化）
- 市场：围绕明确可验证事实事件（有权威判定来源），以虚拟积分池化或 AMM/配池方式形成即时概率。  
- 交易/参与：用户用平台发放的 Insight Points 参与，系统记录持仓与信心水平；结算后更新声誉分（Brier / log score + 贝叶斯收缩）。  
- 声誉呈现：个人档案展示主题化准确率、校准图、历史复盘与认证徽章；证书可导出给招聘/合作方。  
- 学习闭环：每个结算后自动产出复盘与教学内容，促进用户改进判断力。

合规与治理要点（供法律与运营检查）
- 实施“题材黑名单 + 编辑预审”工作流，用户发起市场须标注判定来源并在编辑通过后才开市。  
- 所有“发放积分”行为需可溯且有非货币化理由（签到、贡献、课程通过、平台活动），并公开积分发放规则。  
- 管理员与高权限操作记录须写入审计表并定期公开治理报告摘要。  
- 上线前与重要变化时须取得 PRC 法律书面意见，并把意见要点内置到合规流程中。

变现与商业边界（高层原则）
- 可变现：B2B 数据/仪表盘、企业私有市场与决策服务、教育/认证课程、非博彩类广告与内容分发。  
- 禁止：出售或以任何方式变现预测积分、售卖可影响胜算的特权（例如买入更多下注额）。  
- 盈利侧重长期价值（企业订阅、认证付费），而非短期用户付费刺激。

社区与传播策略（高效导流路径）
- 与知乎类社区、B站内容创作者、高校与行业研究机构建立合作，用“训练营+复盘”与“专家擂台”作为种子活动。  
- 给早期专家与优质创作提供“可导出证书/媒体推荐”作为非金钱激励，带来用户层级效应。  
- 重视社交分享：提供可嵌入的预测卡片、复盘长文与数据图表，便于在微信/知乎传播。

衡量成功的核心 KPI（用于校准方向）
- 用户层：DAU、D7 留存、每用户每周市场参与数、平均预测质量提升（Brier 降低）。  
- 社区质量：通过审核上架的市场比率、被撤销/投诉市场数、作弊封禁数。  
- 商业指标：企业试点数、B2B 收入占比、认证课程付费转化率。  
- 合规指标：律师意见已覆盖的关键场景、审计日志完备率、监管问询响应时间。

执行优先级（0–3 个短期必须做的事）
1. 全平台文案与 UX：把“SIMULATION ONLY — NO REAL MONEY”横幅与 TOS/FAQ 写好并放到显眼位置。  
2. 建立题材黑名单与编辑预审流程（先人工再自动化）。  
3. 设计并实现积分发放矩阵（仅通过非付费渠道获得），以及审计日志结构与保留策略。  
4. 与 PRC 律师约谈并获取书面意见（把产品说明与运营规则交给律师）。

一句话总结（供 AI/产品随时参考）
- 这是一个“知识与声誉驱动”的模拟平台：零现金、零兑换，围绕可验证事件把群体判断可视化、量化并用于学习与企业决策 —— 所有设计都以合规、透明与长期信任为前提。
